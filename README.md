# Selenium Java Automation Framework

## About
This repository showcases automated UI testing using Selenium WebDriver, Java, and TestNG. It includes tests demonstrating browser automation fundamentals and real-world troubleshooting (handling bot detection, dynamic locators).

## Tech Stack
- Java 25
- Selenium WebDriver 4.27.0
- TestNG 7.10.2
- Maven

## Tests Included
- **GoogleTitleTest** — verifies the Google homepage loads with the correct page title
- **OpenCartSearchTest** — searches for a product on the OpenCart demo store and verifies results appear, handling anti-bot detection via ChromeOptions

## Project Structure

src/
├── main/java/org/example/    — exploratory/utility scripts
└── test/java/org/example/    — TestNG automated tests

## Skills Demonstrated
- Selenium WebDriver setup and configuration
- Locator strategies (ID, name, CSS selectors)
- TestNG assertions and test structure
- Handling real-world automation challenges (bot detection, dynamic waits)
